## 介绍

本项目主要是react+antd+react-router+Mobx+Typescript

### `npm dev`

启动本地项目和mock系统

### `npm start`

启动本地项目

### `npm test`

单元测试（未做）

### `npm run build`

打包项目到build文件中

### `npm run eject`

eject 开发环境和打包环境的webpack配置等

### `npm mock`

启动mock系统
